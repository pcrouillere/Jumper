Présentation du projet

Qu'est-ce Jumper ?

La majorité des personnes sauvegardent leurs favoris sans les utiliser par la suite. Parfois trop compliqué d'accès, parfois trop nombreux, les systèmes actuels pour gérer les favoris ne sont pas adaptés.
#Jumper propose un système qui gère les favoris à l'aide de tags, mais aussi de nouveaux systèmes de visualisation de vos favoris préférés, ainsi qu'un système de recherche inspiré du graphe de Venne.


C'est une nouvelle solution pour faciliter vos veilles informatiques, vos recherches parmis vos favoris mais aussi le partage de ces derniers.

Maquettes

L'idée était ici de développer un maximum de fonctionnalités pour permettre à l'utilisateur de re-découvrir le système de favoris. Le but était donc de développer :
- Une page d'accueil regroupant l'ensemble des favoris
- Une page de recherche permettant à l'utilisateur de faire des recherches complexes dans leur ensemble de favoris de manière simplifiée à l'aide d'une représentation d'un diagramme de Venne
- Une page graphe pour montrer les liens entre les tags et favoris
- Une page tableau de bord pour recensser tous les tags et favoris respectifs de l'utilisateur

Recherche simple

L'outil de recherche permet de rechercher les liens via leur titre

vous n'avez plus qu'à cliquer et les sites sont ajoutés automatiquement !
Vous pourrez ensuite revenir les taguer ou bien les retrouver

Rendez-vous sur #Jumper

Fonctionnalité : Graphe

Un nouveau système d'affichage

Le graphe est une représentation de l'ensemble de vos tags
Plus un noeud est grand plus le tag qu'il représente possède de favoris.
Chaque lien représente une liaison entre vos tags : c'est-à-dire qu'ils partagent au moins un site en commun
Le code couleur vous permet de vous rendre compte rapidement des tags qui sont liés entre eux

Affichage

Cliquez sur un noeud et l'ensemble des favoris marqués par ce tag s'affiche !

Fonctionnalité : Recherche

Un système de recherche à l'aide d'un diagramme de Venne

Vous aimeriez retrouver, dans vos favoris, un ensemble de liens concernés par plusieurs tags ?
C'est à présent possible. Il vous suffit de selectionner les tags que vous souhaitez, puis de cliquer sur les zones qui vous intéressent.
Plus la peine de connaître des formules logiques, l'application réfléchi pour vous !

Fonctionnalité : Tableau Bord

Affichage par tag

Ici vous retrouvez l'ensemble de vos tags ainsi qu'un apperçu des sites qu'ils contiennent
Vous pouvez cliquer directement sur un favori ou bien allez consulter l'ensemble des favoris tagués par le tag de votre choix

Choisissez l'ordre d'apparition des tags

Les tags sont affichés selon un ordre qui n'est pas figé
Vous pouvez choisir l'ordre dans lequel ils apparaissent, pour cela il suffit de les déplacer !
L'application se chargera de se souvenir de cet ordre et de vous l'afficher toujours ainsi!

Fonctionnalité : Tableau par tag

Affichage de l'ensemble des liens

Ici vous retrouvez l'ensemble des favoris associés à un tag
Pour chaque favori l'icone de ce dernier est affiché, ainsi que son titre et l'ensemble des tags qu'il possède

Supprimer le tag d'un favori

Il suffit simplement de "détaguer" le favoris à l'aide de l'icone "-" présent

Ajouter des tags à un favori

L'icone "+" vous permet d'accéder directement à l'outil qui gère l'ajout de tags sur un lien.
Vous n'aurez plus qu'à faire glisser les tags que vous souhaitez ajouter sur votre favori et le tour est jouer !

Supprimer un favori

En cliquant simplement sur la poubelle, le lien n'apparaîtera plus dans vos favoris

Le projet

Pourquoi?

Le système de dossier est dépassé, l'usage de tags est de plus en plus répendu sur le Web. Cependant les sites permettant de gérer des favoris ne proposent pas toujours des solutions efficaces
#Jumper permet de trier de manière simple ses favoris, tout en gardant une interface esthétique et simplifiée.
De plus l'outil de recherche avancé, permettant de réaliser des recherches compliquées dans notre ensemble de favoris est un atout majeur.

Notre méthode de travail

Nous avons utilisé Trello pour nous répartir les taches, ainsi que GitHub pour gérer le code

Les technologies

Les technologies que nous avons utilisées sont multiples :
- Le modèle MVC
- Le JEE
- Les Servlet
- HTML5 et CSS3
